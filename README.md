### DigitalOceanSpaces interface to upload and delete files using Node.js (also compatible with AWS S3)

## Features
* Upload files to spaces
* Delete files from spaces

## Technologies
* Node.js
* Typescript
* Multer
* DigitalOcean Spaces
* AWS S3 Sdk
